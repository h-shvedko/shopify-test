## Theme Editor Settings
After uploading, configure in the Theme Editor:
1. **Header** > Logo: Upload `logo-brewmaster.svg` from assets/
2. **Hero Banner** > Background image: Upload `hero-brewing.jpg`
3. **Category Grid** > Add category_card blocks pointing to each collection
4. **Featured Products** > Select a collection to display
